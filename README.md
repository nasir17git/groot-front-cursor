# 친환경 미션 플랫폼 (React 프로토타입)

## 소개
- 탄소절감 미션, 여행코스, 마이페이지, 온보딩 등 주요 기능을 포함한 React 웹앱 프로토타입
- 한글 UI, 더미 데이터 기반, 구조 위주

## 폴더 구조
```
src/
  components/   # 공통 컴포넌트
  pages/        # 주요 화면별 컴포넌트
  App.js        # 라우팅 및 레이아웃
  index.js      # 진입점
```

## 실행 방법
1. 의존성 설치
```
npx create-react-app .
npm install
```
2. src 폴더 내 파일/폴더를 덮어쓰기
3. 개발 서버 실행
```
npm start
```
